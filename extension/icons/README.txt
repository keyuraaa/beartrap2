BearTrap IDS - Extension Icons

Required icon files for the Chrome extension:

1. icon16.png  - 16x16 pixels (toolbar icon)
2. icon48.png  - 48x48 pixels (extension management page)
3. icon128.png - 128x128 pixels (Chrome Web Store)

To add the blue paw print logo:
1. Use an image editor (Photoshop, GIMP, or online tool like Canva/Figma)
2. Resize the paw print image to the sizes above
3. Save as PNG files with transparent background
4. Place them in this folder

Recommended: Use the blue paw print (#0066CC) on transparent background

Quick online tool: https://www.iloveimg.com/resize-image
- Upload your paw print image
- Resize to 16x16, 48x48, and 128x128
- Download and rename to icon16.png, icon48.png, icon128.png
